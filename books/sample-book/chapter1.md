# 第一章：开始使用

这是第一章的内容，介绍如何开始使用。

## 1.1 准备工作

在开始之前，你需要准备以下内容：

1. 一台电脑
2. 文本编辑器
3. 基本的 Markdown 知识

## 1.2 第一个示例

这是一段示例代码：

```javascript
function sayHello(name) {
  console.log(`Hello, ${name}!`);
}

sayHello('World');
```
