# 关于

## 关于 Bookstore

Bookstore 是一个基于 [Docsify](https://docsify.js.org/) 构建的在线书籍阅读网站。

## 特点

- **简单** - 只需要 Markdown 文件，无需复杂的构建过程
- **快速** - 纯前端渲染，无需后端服务器
- **灵活** - 支持自定义主题和插件
- **免费** - 可以免费部署在 GitHub Pages

## 技术栈

- [Docsify](https://docsify.js.org/) - 文档网站生成器
- [GitHub Pages](https://pages.github.com/) - 静态网站托管
